# Smart Custom Keyboard

## 项目简介
这是一个基于 Android 平台的自定义键盘应用项目。

项目的初衷源于对现有手机键盘输入体验的不满，特别是词语匹配的准确性问题。本项目的核心目标是打造一个**基于用户习惯的智能键盘**。它能够学习用户的输入模式，随着使用时间的推移，变得越来越懂你，提供更加精准的词语推荐。

## 核心特性
- **个性化学习**: 记录并分析用户的输入习惯，优先推荐用户常用的词汇。
- **自适应优化**: 随着使用频率的增加，键盘的预测模型会不断进化，提升输入效率。
- **隐私优先**: 所有的学习数据将优先在本地处理，确保用户隐私安全。

## 技术栈
- **语言**: Kotlin
- **UI 框架**: Jetpack Compose
- **构建工具**: Gradle
- **最低 SDK**: Android 11 (API 30)
- **目标 SDK**: Android 15 (API 35)

## 开发计划
- [ ] 基础键盘界面实现 (Compose)
- [ ] 输入法服务 (InputMethodService) 集成
- [ ] 基础输入引擎
- [ ] 用户词库与习惯记录模块
- [ ] 智能预测算法实现

## 如何运行
1. 克隆项目到本地
2. 使用 Android Studio 打开项目
3. 同步 Gradle 依赖
4. 连接 Android 设备或模拟器运行
